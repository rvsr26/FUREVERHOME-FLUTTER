ğŸš€ Getting Started
Prerequisites

Flutter SDK

Code editor (VS Code or Android Studio)

Configured Firebase project

Installation
# Clone the repository
git clone https://github.com/rvsr26/FUREVERHOME-FLUTTER-.git

# Navigate to the project directory
cd FUREVERHOME-FLUTTER-

# Install dependencies
flutter pub get

Environment Setup

Create a .env file in the root directory and add your Gemini API key:

GEMINI_API_KEY=your_api_key_here


Note: The .env file is already listed in .gitignore to protect your key.

Firebase Setup

Generate your own firebase_options.dart file with the FlutterFire CLI:

flutterfire configure


This links your app to your Firebase backend.

Run the App
flutter run
